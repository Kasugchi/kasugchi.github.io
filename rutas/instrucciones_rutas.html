🚛 Guía rápida · Rutas Offline

Una guía sencilla para registrar gastos, kilómetros, tickets y ubicación durante un viaje.

1

Empieza creando o seleccionando un viaje

Usa Nuevo viaje para separar rutas distintas. Por ejemplo:

Algeciras → Lyon · Junio 2026

Todo lo que registres se guardará dentro del viaje seleccionado.

2

Elige el tipo de registro

Pulsa uno de los accesos rápidos antes de guardar:

🚧 Peaje ⛽ Gasolina 📏 Km 🏨 Hospedaje 🍔 Comida 📝 Otros

Otros sirve para cualquier cosa rara: parking, ferry, taller, lavado, etc.

3

Rellena precio, litros y kilómetros

CampoCómo usarlo
PrecioImporte pagado. En Km puede ser 0.
LitrosSolo si has repostado.
KilómetrosEl odómetro del vehículo, no la distancia del tramo.
Para calcular bien el viaje, mete los kilómetros del vehículo al inicio y luego actualízalos durante la ruta.
4

Guarda el registro

Cuando pulses Guardar registro, la app intentará coger el GPS automáticamente.

  • Con internet: guarda dirección real.
  • Sin internet: guarda latitud y longitud.
  • Cuando vuelva internet: usa Completar direcciones pendientes.
Aunque no haya cobertura, el registro se puede guardar y la ubicación se puede completar más tarde si hay coordenadas.
5

Tickets y fotos

Antes de guardar puedes añadir una foto del ticket. Después podrás verla desde el registro con Ver ticket.

Útil para peajes, gasoil, hoteles o comidas.

6

Mapa de ruta

El mapa dibuja los puntos GPS guardados y una línea entre ellos.

  • Los puntos se guardan aunque estés offline.
  • El mapa necesita internet para cargar el fondo de OpenStreetMap.
  • Cuando vuelva internet, los puntos aparecerán dibujados.
7

Resumen

El resumen muestra gasto total, litros, kilómetros, gasolina, peajes, media y coste por km.

La media y el coste/km tienen sentido si los kilómetros se usan como odómetro: km final menos km inicial.
8

Copias y exportaciones

  • Exportar Excel completo: genera un Excel con resumen y viajes.
  • Backup JSON viaje: copia solo el viaje actual.
  • Backup JSON completo: copia todos los viajes.
  • Importar JSON: restaura o fusiona datos.
Haz un Backup JSON completo de vez en cuando. Es tu copia de seguridad más importante.

✅ Uso recomendado en ruta

  1. Crear viaje.
  2. Registrar los kilómetros iniciales con el botón Km.
  3. Registrar gasolina, peajes, comidas u otros gastos durante el viaje.
  4. Adjuntar tickets cuando haga falta.
  5. Al final, registrar los kilómetros finales.
  6. Exportar Excel o Backup JSON completo.
